
Event Management System — A role-based platform (User/Organizer/Admin) for discovering, registering, and managing events with real-time seat availability tracking.

Built with Node.js, Express, and MongoDB, featuring JWT authentication, input validation, and secure error handling for production-ready deploymen

Tech Stack
Node.js • Express.js • MongoDB • JWT • bcryptjs

 Installation
git clone https://github.com/Hayan-eng/EventManagement
cd backend
npm install
cp  .env
npm run dev
